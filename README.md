##Getting started
Create python virtual environment
```
python -m venv venv
```

Activate the virtual environment
```
on Windows
./venv/Scripts/activate
on Linux/macOS
source ./venv/bin/activate
```

Activate the enviroment
```
pip install -r requiraments.txt
```

Create a .env file
```
SERVER_NAME=5000
```

Start the server
```
SERVER_NAME=5000
```
